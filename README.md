# 지뢰찾기

[지뢰찾기 웹사이트](http://minesweepe.com/)

## **학습 목표**

- 객체 지향
- TDD, Test Code 작성
- NextJS app directory 학습
- FxTS

## 실행 방법

1. 개발 환경 실행

```bash
npm install
```

```bash
npm run dev
```

2. 빌드 및 실행

```bash
npm run build
```

```bash
npm run start
```

3. 테스트 실행

```bash
npm run test
```

4. Docker 실행

```bash
docker build -t tag-name .
```

```bash
docker run -p 3000:3000 tag-name
```
